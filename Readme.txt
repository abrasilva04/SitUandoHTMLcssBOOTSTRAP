Site criado no processo de aprendizagem de HTML e CSS, um dos primeiros projetos que fiz usando HTML. Tem um pouco de CSS e dois modais tirados dos modais do Bootstrap. Esse projeto foi feito com o auxílio da professora Lívia que ensinava desenvolvimento front-end. A parte do código que usa os modais foi editada. O código para os templates é livre e está à disposição. Lembrando que o site também conta com uma página de checkout e uma lista para adicionar itens no carrinho, que pode ser manipulada com JavaScript, modificando elementos do DOM.
